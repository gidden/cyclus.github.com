Stub Archetypes¶

This is a collection of archetypes found in the stubs path that comes with Cyclus.

stubs:StubRegion:StubRegion¶

A stub region is provided as a skeleton for the design of new region agents.

Other Info¶

  • Full Archetype Name: stubs::StubRegion
  • Simulation Entity Type: region
  • Interfaces: [cyclus::Region]
  • All Interfaces: [cyclus::Agent, cyclus::Ider, cyclus::Region, cyclus::StateWrangler, cyclus::TimeListener]
